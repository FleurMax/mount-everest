# Georges : Objectif Everest

Georges, 81 ans, s'apprête à gravir le mont Everest en 2027. Cette plateforme documente son entraînement, sa philosophie de vie "L'âge est un chiffre", et permet de soutenir son ascension.

## 🚀 Fonctionnalités
- **Compte à rebours** : Compte à rebours dynamique jusqu'au 1er avril 2027.
- **Journal d'entraînement** : Suivi de sa préparation physique.
- **Guide de vie** : Conseils sur la nutrition, le sommeil et la mobilité pour les seniors.
- **Accessibilité** : Design conçu pour un public de 60 à 90 ans (grandes polices, contraste élevé).

## 🛠️ Stack Technique
- **Framework** : React.js
- **Build Tool** : Vite
- **Styling** : Vanilla CSS 
- **Icons** : Emojis & Custom CSS

## 📋 Installation
1. Clonez le dépôt : `git clone https://github.com/FleurMax/mount-everest.git`
2. Installez les dépendances : `npm install`
3. Lancez le serveur de développement : `npm run dev`
